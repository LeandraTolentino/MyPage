Linear Equations

What is a Linear Equation?

A linear equation is an equation where the graph is a line with constant slope. It takes the form mx + c, where m is the slope and c is a constant.

Remember from primary school that two sides of an equation must be equal.

Linear equations follow the same rule. However, remember that a variable can be any number! So x = 5 is okay.

However x = x + 5 is not okay, because if we subtract the x we get 0 = 5. This is nonsense!
